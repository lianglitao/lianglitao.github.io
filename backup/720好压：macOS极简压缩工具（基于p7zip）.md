720好压是一款专为macOS设计的极简压缩/解压工具，核心基于开源的p7zip（7-Zip的macOS实现），仅3个文件（2个功能文件+1个开源协议），无冗余功能，专注解决macOS下压缩/解压的核心需求。

## 🌟 核心特性
- 极简轻量：仅2个文件，无安装包，直接可执行
- 格式全面：支持7z/ZIP/RAR等主流压缩格式
- 离线可用：无网络请求，100%离线操作
- 开源合规：遵循p7zip的LGPL开源协议

## 📦 快速安装
### 1. 安装底层依赖（p7zip）
首先通过Homebrew安装p7zip（macOS下7-Zip的正统实现）：
```bash
brew install p7zip
